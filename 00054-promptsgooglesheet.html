⚙️ Workflow Technique : Art Bible

  • Structure Google Sheet : Le script doPost génère automatiquement 9 colonnes (ID, Thème, Artiste, Prompt +, Prompt -, Seed, Lien Image, Note, Date) et les onglets par catégorie.
  • Google Apps Script :
    • Accès via Extensions > Apps Script.
    • Déploiement via Gérer les déploiements > Nouvelle version (pour conserver l'URL /exec stable).
  • Google Colab :
    • Outil de transmission : colab.google.
    • Variable URL_SHEET : Coller l'URL /exec.
    • Action : Coller le dictionnaire Python et exécuter (Play).
  • Hébergement Image (Confidentialité) : Utilisation d'URL kDrive (Infomaniak). Note : S'assurer que le "Lien de partage" est public pour l'affichage dans le Sheet.
